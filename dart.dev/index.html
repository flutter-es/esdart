Un lenguaje accesible, portátil y productivo para aplicaciones de alta calidad en cualquier plataforma

Dart es gratuito y de código abierto  

Accesible
Desarrolle con un lenguaje de programación consistente, conciso y fuertemente tipado que ofrezca características modernas como seguridad nula y coincidencia de patrones.
Desarrollo
productivo
Realice cambios de forma iterativa: use la recarga activa para ver el resultado instantáneamente en su aplicación en ejecución.
Portátil y rápido
en todas las plataformas
Compílelo en código de máquina ARM, x64 y RISC-V para dispositivos móviles, computadoras de escritorio y backend. O bien, compílelo en JavaScript y WebAssembly para la web.
Accesible
  • Async-await maduro y completo para interfaces de usuario que contienen código controlado por eventos, combinado con concurrencia basada en aislamiento.
  • Escriba código seguro y conciso utilizando funciones como seguridad nula, recopilación de valores if, y coincidencia de patrones.
  • A consistent programming language, with an easy to learn and familiar syntax.
Productive Development
  • Make changes to your source code iteratively, using hot reload to instantly see the effect in the running app.
  • Write code using a flexible type system with rich static analysis and powerful, configurable tooling.
  • Do profiling, logging, and debugging with your code editor of choice.
Fast on all platforms
  • AOT-compile apps to native machine code for instant startup.
  • Target the web with complete, mature, fast compilers for JavaScript & WebAssembly.
  • Run backend code supporting your app, written using a single programming language.

Desarrollo productivo

  • Realice perfiles, registros y depuraciones con el editor de código de su elección.
Rápido en todas las plataformas
  • AOT compila aplicaciones en código de máquina nativo par un inicio instantáneo.
  • Apunte a la web con compiladores completos, maduros y rápidos para JavaScript y WebAssembly.
  • Ejecute código backend que respalde su aplicación, escrito con un solo lenguaje de programación.
